Hexo+Github博客

　　自从今年9月份，买了.com域名以来，便开始尝试搭建一个真正属于自己的网站，顺便把自己所学的web知识做个记录。
　　一开始我是使用wordpress搭建网页，但因为后面还需要去淘宝租用香港的虚拟主机（免备案），所以试用一个月后也没有动力更新网站样式，更不要说写博客。直到一天偶然翻到某大神的博客，里面推荐了一种farbox+Dropbox网盘免费空间的办法，便学了起来，却发现国产farbox已经不做免费生意了，需升级成bitcron。看回大神网站的底部，写着powered by hexo，好了，原来又换成hexo弄，遂开始Hexo+github之旅。
　　就这样一边兼顾设计任务，一边跌跌撞撞的成功的“克隆”了大神的博客主题，这篇文章也由此诞生。下面将记录下创建过程所遇到的问题。
　　
　 1. 需要安装的工具，注册的，不细说。说下重点，git是具体到每个文件夹的，所以要cd选择好目录，或者直接在该目录下git bash here。
　 2. 为什么域名已经绑定了github，还是显示404？在github仓库的setting也要填上自己的域名地址。
　 3. 为什么本地hexo显示正常，自己域名输入却还是显示github的页面?hexo没有上传成功，成功的话，github也会更新上传文件。
　 4.要修改主题里_config.yml的标题，必须复制之后再修改，不能直接在后面添加。
